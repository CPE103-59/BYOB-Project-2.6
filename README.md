####Control
* Press 'A' to fire
* Use 'Arrow Keys' to control spaceship
  * 'UP Arrow' for increase speed
  * 'Left Arrow' to ~~turn left~~ rotate counter-clockwise
  * 'Right Arrow' to ~~turn right~~ rotate clockwise
